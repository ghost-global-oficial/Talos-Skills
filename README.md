# Talos-Skills

Repositorio de skills do Talos - FlowLabs by Ghost Systems.

## Estrutura

```
├── skills-config.json    # Configuracao central das skills
├── Integrations/         # Skills de integracao externa
├── MCPs/                # Model Context Protocol servers
├── Templates/           # Templates de skills
├── question-card/       # Card interativo de perguntas
├── web-search/          # Pesquisa web com card de resultados
├── table/               # Tabelas estruturadas inline
└── mini-vscode/         # Mini editor de codigo
```

## Uso

As skills sao carregadas automaticamente pelo FlowLabs a partir do `skills-config.json`.
